# Changelog

## 1.0.0 (2025-03-22)

### Features

- 🎉 Initial release
- QR code login with auto-refresh (up to 3 times)
- Long-poll message receiving (like Telegram Bot API)
- Send/receive text, image, video, file, and voice messages
- AES-128-ECB media encryption/decryption for CDN
- Automatic credential persistence
- Typing indicators (send/cancel)
- Event-driven API (`bot.on('message', ...)`)
- Full TypeScript type declarations
- Zero external dependencies
